This calculator is based on a simple computer calculator which performs several tasks like addition,subtraction,multiplication and some other tasks. This is designed with assembly language.Functions of this calculator are given below with their proper description.

Function Description
Addition (+) The addition function add two numbers and show the result.
Subtraction (-) The subtraction function subtract two numbers and show the result.If we subract a greater number from a lower number the result will be in negative
Multiplication (*) The multiplication function multiply two numbers and show the result.
Division (/) The division function divide the divident by the divisor and show the result.
Mod (%) The Mod function will show the remainder after any divison.
Square (^) The Square Function will square the number entered

Description of Addition

1.Take an input. Saved the value to a variable.
2.Take another input.saved the value to a variable.
3.Added the inputted value. Saved the value to a register AX.
4.The outputproc was called.
